# Instructions to Run the Report Generator

## Overview
The program automatically discovers CSV files and photo files in the same folder and generates comprehensive battery analysis reports in PDF and Word formats.

## How to Run
1. Place all CSV data files and photo files in the same folder as `report_generator.py`.
2. Open a terminal/PowerShell in that folder.
3. Activate the virtual environment (if needed):
   ```
   .\venv\Scripts\activate
   ```
4. Run the script:
   ```
   python report_generator.py
   ```
5. Reports will be generated in a `reports/` subfolder automatically.

## Processing Time
Processing may take 5-15 minutes depending on your computer and data size. Large datasets (1M+ rows) require more time for:
- CSV parsing
- Data resampling
- Plot generation
- Report compilation

**Be patient and let the script complete** - you will see progress messages indicating what's happening.

## Output Files
Reports are saved in the `reports/` folder with timestamps:
- `battery_report_YYYYMMDD_HHMMSS.pdf` - PDF report
- `battery_report_YYYYMMDD_HHMMSS.docx` - Word report

## Report Sections
The generated reports include:
1. **Electrical Performance** - Voltage range, peak power, average power (per shift)
2. **Energy Efficiency** - Distance covered, total energy consumed (per shift)
3. **SOC Analysis** - Initial/Final SOC and depletion (per shift)
4. **Ride Mode Analysis** - Performance by ride mode with tables and plots
5. **Thermal Performance** - Temperature analysis from sensors
6. **Charge/Discharge Profile** - Battery status distribution analysis

## Troubleshooting
- **Missing dependencies**: Run `pip install -r requirements.txt`
- **No CSV files found**: Ensure .csv files are in the same folder as the script
- **Reports not generating**: Check the console for error messages and ensure you have write permissions
